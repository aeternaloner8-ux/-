# ВЕЛОРА — модульные дома

Лендинг компании по строительству модульных домов «ВЕЛОРА».
Самодостаточная статическая страница (HTML со встроенными стилями и скриптами), готовая к публикации через GitHub Pages.

## Файлы
- `index.html` — главная страница (точка входа для GitHub Pages)
- `velora-homes.html` — та же страница под исходным именем

## Локальный просмотр
Просто открой `index.html` в браузере. Или запусти простой сервер:

```bash
python3 -m http.server 8000
```

затем открой http://localhost:8000

## Публикация (GitHub Pages)
1. Репозиторий на GitHub → **Settings → Pages**
2. **Source:** Deploy from a branch
3. **Branch:** `main`, папка `/ (root)` → **Save**
4. Через 1–2 минуты сайт будет доступен по адресу:
   `https://aeternaloner8-ux.github.io/-/`
